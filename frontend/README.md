# PC Finder Frontend

Vercel/Stripe-inspired dark UI + gradients, but kept to **very few code files**.

## Files you'll touch
- `app/page.tsx` (landing + search + results)
- `app/researcher/[id]/page.tsx` (detail page)
- `lib/api.ts` (API + types)
- `styles/globals.css`

## Run
```bash
cp .env.example .env.local
npm install
npm run dev
```

Open http://localhost:3000
